================================================================================
GUÍA: CÓMO AGREGAR NUEVOS CAMPOS A LA TABLA INVENTARIO
================================================================================

Objetivo: Agregar tres nuevos campos al inventario:
  • descripcion (STRING) - Descripción detallada del insumo
  • stock_minimo (INT) - Cantidad mínima recomendada (para alertas)
  • fecha_ingreso (DATETIME) - Fecha cuando se agregó el insumo

Estos cambios permitirán registrar un nuevo insumo con los nuevos campos
usando el formulario de registro en el frontend, y guardarlos en la BD.

================================================================================
1. AGREGAR CAMPOS EN LA BASE DE DATOS
================================================================================

OPCIÓN A: Si es la PRIMERA VEZ (tabla vacía o en construcción)

Archivo: Cualquier script SQL que cree la tabla inventario
Busca la sentencia CREATE TABLE inventario y localiza esta línea:
    CHECK (cantidad_disponible >= 0 AND cantidad_disponible <= cantidad_total)
  ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

Reemplázala por:
    CHECK (cantidad_disponible >= 0 AND cantidad_disponible <= cantidad_total),
    descripcion VARCHAR(500),
    stock_minimo INT DEFAULT 5,
    fecha_ingreso DATETIME DEFAULT CURRENT_TIMESTAMP
  ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;


OPCIÓN B: Si YA EXISTEN DATOS EN LA TABLA

Ejecuta estos comandos en phpMyAdmin (ir a Operaciones > SQL):
  ALTER TABLE inventario ADD COLUMN descripcion VARCHAR(500);
  ALTER TABLE inventario ADD COLUMN stock_minimo INT DEFAULT 5;
  ALTER TABLE inventario ADD COLUMN fecha_ingreso DATETIME DEFAULT CURRENT_TIMESTAMP;

✓ QUÉ SIGNIFICAN:
  • descripcion VARCHAR(500) → Campo de texto de hasta 500 caracteres
  • stock_minimo INT DEFAULT 5 → Campo numérico con valor por defecto 5
  • fecha_ingreso DATETIME DEFAULT CURRENT_TIMESTAMP → Fecha automática de creación
