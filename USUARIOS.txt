================================================================================
GUÍA: CÓMO AGREGAR NUEVOS CAMPOS A LA TABLA USUARIOS
================================================================================

Objetivo: Agregar dos nuevos campos a los usuarios:
  • torre (INT) - El número de la torre del edificio
  • apellido (STRING) - El apellido del titular

Estos cambios permitirán registrar un nuevo usuario con los nuevos campos
usando el formulario de registro en el frontend, y guardarlos en la BD.

================================================================================
1. AGREGAR CAMPOS EN LA BASE DE DATOS
================================================================================

OPCIÓN A: Si es la PRIMERA VEZ (tabla vacía o en construcción)

Archivo: Cualquier script SQL que cree la tabla usuarios
Busca la sentencia CREATE TABLE usuarios y localiza esta línea:
    fecha_registro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

Reemplázala por:
    fecha_registro TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    apellido VARCHAR(100),
    torre INT
  ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;


OPCIÓN B: Si YA EXISTEN DATOS EN LA TABLA

Ejecuta estos comandos en phpMyAdmin (ir a Operaciones > SQL):
  ALTER TABLE usuarios ADD COLUMN apellido VARCHAR(100);
  ALTER TABLE usuarios ADD COLUMN torre INT;

✓ QUÉ SIGNIFICAN:
  • apellido VARCHAR(100) → Campo de texto de hasta 100 caracteres
  • torre INT → Campo numérico entero (ej: 1, 2, 3, 4, etc.)
